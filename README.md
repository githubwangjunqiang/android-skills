# Android Skills

Claude Code Skills 规范文件集合，用于 Android 项目开发。

## 包含的 Skills

| 技能 | 说明 |
|------|------|
| `android-project-overview` | 总纲：技术栈、强制规范、包结构、代码风格、AndroidManifest、构建命令 |
| `android-mvi-compose` | MVI 基础设施、Intent/State/Effect 模板、ViewModel 模板、Compose 状态页组件 |
| `android-local-storage` | MMKV 键值存储、Room 数据库、存储选型 |
| `android-utils-core` | 工具函数（Toast、JSON、时间、DP/PX）、图片加载、日志系统、屏幕适配 |
| `android-advanced-dev` | Repository 模式、全局异常处理、多语言、深色模式、动画、单元测试 |
| `android-build-publish` | buildSrc 模块、版本号自增管理、蒲公英上传、APK 命名 |
| `android-legacy-view` | Activity 启动模板、传统 View 状态页容器、View-Compose 互操作 |
| `android-coroutines` | Kotlin 协程规范。协程提供者、作用域管理、异常处理、线程切换 |
| `android-network` | 网络请求架构。OkHttp 扩展函数、请求规范、响应处理、全局错误拦截 |

## 使用方式

将 Skills 目录复制到 `~/.claude/skills/` 目录下即可在 Claude Code 中使用。
